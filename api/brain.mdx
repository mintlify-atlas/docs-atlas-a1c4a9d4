## Exchange

A dataclass representing a single Q&A turn in a conversation.

```python
@dataclass
class Exchange:
    user_text: str
    assistant_text: str
    image_base64: str | None = None
    searches: list[dict] = field(default_factory=list)
    notes_file_changed: bool = False
```

<ResponseField name="user_text" type="str">
  The user's question
</ResponseField>

<ResponseField name="assistant_text" type="str">
  Claude's response
</ResponseField>

<ResponseField name="image_base64" type="str | None">
  Base64-encoded camera image if included
</ResponseField>

<ResponseField name="searches" type="list[dict]">
  List of web searches performed during this turn
</ResponseField>

<ResponseField name="notes_file_changed" type="bool">
  Whether notes were modified during this exchange
</ResponseField>
