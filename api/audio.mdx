## VoiceActivatedRecorder

Continuously listens and uses webrtcvad to detect speech boundaries.

### Constructor

```python
VoiceActivatedRecorder(
    on_speech_start: Callable[[], None],
    on_speech_end: Callable[[bytes], None],
    on_speech_discard: Callable[[str], None] | None = None,
    sample_rate: int = SAMPLE_RATE,
    sensitivity: int = 2,
    silence_timeout: float = 1.5,
    min_voiced_ratio: float = 0.25,
    min_voiced_frames: int = 5,
    min_duration: float = 0.3,
    min_rms_dbfs: float = -45.0,
    min_voiced_run_frames: int = 6,
    device: int | None = None,
)
```

<ParamField path="on_speech_start" type="Callable[[], None]" required>
  Callback invoked when speech begins
</ParamField>

<ParamField path="on_speech_end" type="Callable[[bytes], None]" required>
  Callback invoked with WAV bytes when speech ends
</ParamField>

<ParamField path="on_speech_discard" type="Callable[[str], None] | None" default="None">
  Callback invoked with reason when utterance is discarded
</ParamField>

<ParamField path="sample_rate" type="int" default="16000">
  Audio sample rate in Hz
</ParamField>

<ParamField path="sensitivity" type="int" default="2">
  WebRTC VAD sensitivity (0-3, higher = more aggressive filtering)
</ParamField>

<ParamField path="silence_timeout" type="float" default="1.5">
  Seconds of silence before finalizing speech
</ParamField>

<ParamField path="min_voiced_ratio" type="float" default="0.25">
  Minimum ratio of voiced frames to total frames
</ParamField>

<ParamField path="min_voiced_frames" type="int" default="5">
  Minimum number of voiced frames required
</ParamField>

<ParamField path="min_duration" type="float" default="0.3">
  Minimum utterance duration in seconds
</ParamField>

<ParamField path="min_rms_dbfs" type="float" default="-45.0">
  Minimum RMS loudness in dBFS
</ParamField>

<ParamField path="min_voiced_run_frames" type="int" default="6">
  Minimum contiguous voiced run of 30ms frames
</ParamField>

<ParamField path="device" type="int | None" default="None">
  Input device index, or None for system default
</ParamField>
